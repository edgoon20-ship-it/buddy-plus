# 버디버디플러스 — Firebase 설정 가이드

## 1단계: Firebase 프로젝트 만들기

1. [Firebase Console](https://console.firebase.google.com) 접속 (Google 계정 필요)
2. **"프로젝트 추가"** 클릭
3. 프로젝트 이름 입력 (예: `buddy-plus`)
4. Google Analytics는 선택사항 — 건너뛰어도 됨
5. 프로젝트 생성 완료

---

## 2단계: Firestore 데이터베이스 활성화

1. 좌측 메뉴 **"Firestore Database"** 클릭
2. **"데이터베이스 만들기"** 클릭
3. **"테스트 모드로 시작"** 선택 (30일간 무료 읽기/쓰기 허용)
4. 위치: `asia-northeast3 (서울)` 선택 후 완료

### ⚠️ 30일 후 보안 규칙 설정 (필수)

30일 뒤에는 Firestore > 규칙 탭에서 아래 내용으로 교체하세요:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /events/{eventId} {
      allow read:   if true;
      allow create: if true;
      allow update: if true;
      allow delete: if true;
    }
  }
}
```

## 3단계: 앱 등록 및 설정값 복사

1. 프로젝트 홈에서 **`</>`** (웹 앱 추가) 클릭
2. 앱 닉네임 입력 (예: `buddy-plus-web`) 후 등록
3. Firebase SDK 설정값이 표시됨. 아래처럼 생겼어요:

```javascript
const firebaseConfig = {
  apiKey: "AIza...",
  authDomain: "buddy-plus-xxxxx.firebaseapp.com",
  projectId: "buddy-plus-xxxxx",
  storageBucket: "buddy-plus-xxxxx.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abcdef..."
};
```

4. `buddy-plus.html` 파일을 텍스트 편집기로 열고  
   파일 상단의 `FIREBASE_CONFIG` 부분을 위 값으로 교체

---

## 4단계: 관리자 비밀번호 변경

`buddy-plus.html` 파일에서 아래 줄을 찾아 원하는 비밀번호로 변경:

```javascript
const ADMIN_PW = "bbplus2025"; // ← 여기를 바꾸세요
```

관리자들에게만 공유해주세요.

## 5단계: 호스팅 (링크 공유용)

### 옵션 A — GitHub Pages (무료, 가장 간단)

1. [GitHub](https://github.com) 계정 생성
2. 새 저장소 만들기 (예: `buddy-plus`)
3. `buddy-plus.html` 파일을 `index.html`로 이름 바꿔 업로드
4. 저장소 Settings → Pages → Source: `main` 브랜치 선택
5. 몇 분 후 `https://[아이디].github.io/buddy-plus` 링크 생성!

### 옵션 B — Firebase Hosting (Firebase와 완벽 통합)

Firebase Console에서 "Hosting" 메뉴 → "시작하기" 따라가면 됩니다.  
링크 형태: `https://buddy-plus-xxxxx.web.app`

### 옵션 C — Netlify Drop (가장 빠름, 드래그앤드롭)

1. [app.netlify.com/drop](https://app.netlify.com/drop) 접속
2. `buddy-plus.html` 파일을 그냥 드래그 앤 드롭
3. 즉시 링크 생성! (예: `https://amazing-name-123.netlify.app`)

## 주요 기능 정리

| 기능 | 설명 |
|------|------|
| 모임 발제 | 누구나 등록 가능. 날짜·장소·최대인원·발제자 기재 |
| 참가 신청 | 이름 + 학년 입력으로 신청 |
| 신청자 명단 | 카드에서 탭으로 열람 가능 |
| 모임 삭제 | 발제 시 받은 6자리 코드 또는 관리자 비밀번호 |
| 탭 구분 | 예정된 모임 / 지난 모임 자동 분류 |
| 실시간 반영 | Firebase 덕분에 새로고침 없이 자동 업데이트 |
